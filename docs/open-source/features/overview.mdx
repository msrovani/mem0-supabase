# Features

<CardGroup cols={2}>
  <Card title="Universal AI Access (MCP)" icon="plug" href="/open-source/features/mcp">
    Connect Mem0 as a memory server for Claude Desktop, Cursor, and Continue IDEs.
  </Card>
  <Card title="Temporal Memory" icon="clock-rotate-left" href="/open-source/features/temporal">
    Time-travel queries to recall what the agent knew at any point in history.
  </Card>
  <Card title="Graph Memory" icon="network-wired" href="/open-source/features/graph-memory">
    Entity relationships and knowledge graphs using Supabase Graph-on-Postgres.
  </Card>
  <Card title="LangGraph Native" icon="microchip" href="/open-source/features/langgraph">
    Persistent checkpoints for stateful LangGraph agent swarms.
  </Card>
  <Card title="Multimodal Support" icon="image" href="/open-source/features/multimodal-support">
    Store and process images and documents via Supabase Storage.
  </Card>
  <Card title="Hybrid Search" icon="magnifying-glass" href="/open-source/features/hybrid-search">
    Vector + Keyword search using RRF (Reciprocal Ranked Fusion).
  </Card>
